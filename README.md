## The Stack
* React 18
* Vite
* Material UI
* Socket IO
* Apollo Client

## How to run the project
1. `npm install`
2. `npm run dev`
3. Go to `localhost:5173`
4. Log in with Google for any client user. You need a spetial account to log in as a teacher.

## How to build for production
1. `npm run build`
2. `npm run preview`
3. Go to `localhost:3000`
